Έναρξη του παιχνιδιού τρέχοντας το αρχείο main.py
Ερωτήσεις στον χρήστη:
    - Αν θέλει να παίξει πρώτος (η απάντηση θα είναι yes ή no)
    - Να δoθεί το max depth (πρέπει να είναι integer >= 1)
    - Όταν είναι η σειρά του χρήστη, δίνει πρώτα τη σειρά και μετά τη γραμμή (ξεκινούν από το 0)

Χρήση του genetic_determine_weights
    - Αριθμός γενεών (πρέπει να είναι int)
    - Αριθμός παικτών (πρέπει να είναι ζυγός αριθμός)
    - Το μέγιστο βάθος
    - Ποιά συνάρτηση reproduce να χρησιμοποιήσεo

Χρήση του determine_weights: 
    - Αριθμός τουρνουά (πρέπει να είναι δύναμη του 2)
    - Αριθμός αγωνιζόμενων (πρέπει να είναι δύναμη του 2)
    - Αριθμός βαρών ανά σετ βαρών
